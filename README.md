# Pak64.map
G〇ogleMapみたいなパックセットの試作品です。
pak64のアドオンの導入が可能です。


# 開発用
## Pak化
予めmakeobjにパスを通すか、このディレクトリに設置してください。

```
npm install
npm run build
```
distディレクトリにジャンルごとにマージされたpakが生成されます。
